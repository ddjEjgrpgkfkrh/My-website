  <ul>
    <li>원재료: 수삼(Panax ginseng C.A. Meyer)을 증기 또는 기타방법으로 쪄서 익혀 말린 홍삼</li>
    <li>제조방법: 
      <ul>
        <li>분말화하여 제조</li>
        <li>또는 물/주정(혼합물 포함)으로 추출·여과·농축 또는 발효</li>
      </ul>
    </li>
    <li>기능성분 함량: 진세노사이드 Rg1, Rb1, Rg3 합계 2.5 mg/g 이상</li>
    <li>제조 시 유의사항: 4년근 이상 인삼근 사용, 병든 부분 제거 등</li>
  </ul>
</div>

<div class="section">
  <h2>규격</h2>
  <ul>
    <li>성상: 고유의 색택과 향미, 이미·이취 없음</li>
    <li>진세노사이드 합계: 원료성 제품은 표시량 이상, 최종제품은 표시량의 80% 이상</li>
    <li>세균수: 1 mL당 3,000 이하(농축액)</li>
    <li>대장균군: 음성</li>
  </ul>
</div>

<div class="section">
  <h2>최종제품 요건</h2>
  <ul>
    <li>기능성 내용: 면역력 증진, 피로 개선, 혈소판 응집 억제를 통한 혈액 흐름, 기억력 개선, 항산화, 갱년기 여성 건강 도움</li>
    <li>일일 섭취량:
      <ul>
        <li>면역력·피로: 3~80 mg</li>
        <li>혈액 흐름·기억력·항산화: 2.4~80 mg</li>
        <li>갱년기 여성 건강: 25~80 mg</li>
      </ul>
    </li>
    <li>섭취 시 주의사항: 당뇨치료제·혈액항응고제 복용 시 주의, 알레르기 체질 주의, 이상 시 전문가 상담</li>
  </ul>
</div>

<div class="section">
